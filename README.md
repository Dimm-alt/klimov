# Movie Library / Личная кинотека

**Автор:** Артем Климов

## Описание

Простое desktop-приложение для ведения личной коллекции фильмов.  

Программа написана на Python с использованием встроенной библиотеки Tkinter.

## Возможности

 - Сброс фильтров одной кнопкой
- Автоматическое сохранение в `movies.json`
- Загрузка данных при запуске
- Добавление фильмов с проверкой корректности ввода
- Отображение списка фильмов в таблице
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по году (точное совпадение)

## Требования

- Python 3.6 или выше
- Стандартные библиотеки: `tkinter`, `json`, `os`
